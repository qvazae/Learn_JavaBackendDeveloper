## Что реализовано

- Очередь транзакций (`LinkedBlockingQueue`) + несколько касс (`Thread`) обрабатывают операции асинхронно.
- Обмен валют и переводы между клиентами с конвертацией через общую базу курсов.
- ScheduledThreadPoolExecutor обновляет курсы раз в секунду с небольшим дрейфом.
- Паттерн Observer: любой наблюдатель получает события (используется `ConsoleLogger` с метками времени).
- Потокобезопасность: клиенты блокируются через `ReentrantLock`, глобальные структуры — `ConcurrentHashMap`.

Приложение стартует три кассы, регистрирует клиентов, ставит транзакции в очередь, обновляет курсы и корректно завершает работу через `Bank.shutdown()`.
